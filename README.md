
# Find-Rommie-Web

Proyecto web para encontrar habitaciones y roomies confiables en Colombia y México.

## Tecnologías principales

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Instalación y uso

```sh
git clone https://github.com/NeuroNet1/Find-Rommie-Web.git
cd Find-Rommie-Web
npm install
npm run dev
```

## Despliegue

Puedes desplegar el proyecto en Vercel, Netlify o tu plataforma preferida.
